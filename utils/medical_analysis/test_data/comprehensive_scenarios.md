# 医疗分析综合场景虚拟数据

> 本文件包含 5 个完整的临床模拟场景，每个场景可同时调用多个 Agent Tool 进行综合分析。
> 数据基于真实临床参考范围（WS/T 405-2012、三甲医院检验标准）生成。

**患者信息**：王强，男，45岁，职员
**主诉**：发热3天，伴咳嗽、咽痛

- **综合分诊**: 中-高风险，建议感染科/呼吸内科就诊

- **综合分诊**: 高风险，建议内分泌科+肾内科联合就诊

- **综合分诊**: 高风险，建议消化内科/肝病科就诊，戒酒
